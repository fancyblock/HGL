| 源自我去年12月开始写的一个库HGL，当时想写个用于写Demo方便的东西，以避开WIN32编程那一套麻烦的东西； |
| 而且一开始的HGL只是一个2D的库，当时在书上看到透视的计算，便在HGL里添加了一个画3D线条的函数，后来写 |

|                                                                                                    |

| 今年9月，下定决心开始从头开始写HGL，目标是写成一个软件渲染的引擎，利用部分业余时间断断续续进行，边 |
| 学习图形学边做这个东西，希望能坚持到底。                                                           |
